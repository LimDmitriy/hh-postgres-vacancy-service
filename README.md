# HH.ru Vacancy Manager

Программа для работы с API HeadHunter и PostgreSQL. Позволяет получать информацию о компаниях, их вакансиях и хранить данные в базе, а также выполнять базовые аналитические запросы.

## Возможности

- Добавление новых компаний и вакансий по ключевому слову.
- Просмотр списка всех компаний и количества вакансий у каждой.
- Просмотр списка всех вакансий с деталями (название компании, название вакансии, зарплата, ссылка).
- Расчет средней зарплаты по всем вакансиям.
- Просмотр вакансий с зарплатой выше средней.
- Поиск вакансий по ключевым словам.
- Выход из программы.

## Требования

- Python 3.12+
- PostgreSQL
- Библиотеки Python:
  - `psycopg2-binary`
  - `python-dotenv`
  - `requests` 
- Для проверки кода и автоформатирования (опционально):
  - `black`
  - `isort`
  - `flake8`
  - `mypy`
